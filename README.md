# LimitCollector

Задание:
Необходимо реализовать коллекцию, которая имеет фиксированный размер.

Размер задаётся:

по умолчанию 16
по указанию пользователя.
Размер коллекции после инициализации меняться не может.

Каждый новый элемент добавляется в конец. После того, как количество элементов 
достигло максимума - при добавлении нового элемента коллекция "сдвигается" влево.

1-й элемент удаляется, а каждый следующий смещается на один элемент влево.

Результатом выполнения задания должен быть репозиторий с коллекцией, классом, тестирующим
данную коллекцию, описанием задания и README файлом с инструкцией по запуску приложения.

Решением является созданный класс LimitCollector, и класс Main, в котором создается новый объект 
LimitCollector размером 25 ячеек. Он рандомно заполняется числами от 1 до 100 и выводится на экран.
Затем добавляется еще 10 элементов, и при выводе можно увидеть, что коллекция заполняется в соответствии с условием.
